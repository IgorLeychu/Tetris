Task:
Based on the example from the src_sample folder, implement a Tetris game.
It is mandatory to use the BaseApp class of the example.
BaseApp allows you to make a graphical output of the program, handle the keyboard, and organize the main program loop.
The implemented tetris demo can be viewed in the tetris_sample folder.

Mandatory requirements:
1. The presence of preview of the next figure
2. Controls (left/right arrows), figure acceleration (down arrow), figure rotation (space bar)
3. Width of the playing field - 15 symbols
4. The size of the window should be big enough to visually fit the whole playing field.
